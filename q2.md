## 1. Dependencies

| Dependency | Exists? |
| ---------- | ------- |
| A → B      | No      |
| A → C      | No      |
| B → A      | No      |
| B → C      | Yes     |
| C → A      | No      |
| C → B      | No      |
| (B, C) → A | No      |

So the only dependency that may exist is **B → C**.

## 2. Primary Key

Since **B → C**, the best primary key is **(A, B)**.

## 3. Highest Normal Form

The table is in **1NF** only.

It is not in 2NF because `C` depends only on `B`, which is part of the composite key `(A, B)`.
