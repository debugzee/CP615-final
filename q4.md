## (1) SQL Query

Since the university names are inconsistent, we can standardize them first using `CASE` and then join with the `University` table.

```sql
SELECT COUNT(*) AS total_students
FROM (
    SELECT
        StudentID,
        Name,
        Major,
        CASE
            WHEN University = 'WLU' THEN 'Wilfrid Laurier University'
            WHEN University = 'YorkU' THEN 'York University'
            WHEN University = 'York university' THEN 'York University'
            WHEN University = 'UofWaterloo' THEN 'University of Waterloo'
            WHEN University = 'NUS' THEN 'National University of Singapore'
            WHEN University = 'Standford University' THEN 'Stanford University'
            ELSE University
        END AS UniversityName
    FROM Student
) S
JOIN University U
    ON S.UniversityName = U.UniversityName
WHERE S.Major = 'Computer Science'
  AND U.Country = 'Canada';
```

### Result

The answer is:

**2**

## (2) Program for approximate / fuzzy join

A simple approach is:

1. Sort both tables by university name.
2. Standardize names by making them lowercase and removing extra spaces.
3. Compare names during the join.
4. If two names are not exactly the same, use fuzzy matching to check if they are similar.
5. If similarity is high enough, treat them as a match.

### Simple Python example

```python
from difflib import SequenceMatcher

students = [
    ("2001", "Emily Zhang", "Computer Science", "WLU"),
    ("2002", "John Park", "Computer Science", "YorkU"),
    ("2003", "Tina Wang", "Software Engineering", "UofWaterloo"),
    ("2004", "Ahmed Ali", "Information Systems", "York university"),
    ("2005", "Maria Gomez", "Business Analytics", "NUS"),
    ("2006", "Leo Martin", "Cybersecurity", "Standford University")
]

universities = [
    ("Wilfrid Laurier University", "Canada"),
    ("York University", "Canada"),
    ("University of Waterloo", "Canada"),
    ("Stanford University", "United States"),
    ("National University of Singapore", "Singapore")
]

mapping = {
    "wlu": "wilfrid laurier university",
    "yorku": "york university",
    "york university": "york university",
    "uofwaterloo": "university of waterloo",
    "nus": "national university of singapore",
    "standford university": "stanford university"
}

def normalize(name):
    name = name.strip().lower()
    return mapping.get(name, name)

def similar(a, b):
    return SequenceMatcher(None, a, b).ratio()

count = 0

for sid, name, major, uni in students:
    student_uni = normalize(uni)

    for uni_name, country in universities:
        official_uni = normalize(uni_name)

        if student_uni == official_uni or similar(student_uni, official_uni) > 0.8:
            if major == "Computer Science" and country == "Canada":
                count += 1
            break

print("Computer Science students in Canadian universities:", count)
```

### Output

**Computer Science students in Canadian universities: 2**
